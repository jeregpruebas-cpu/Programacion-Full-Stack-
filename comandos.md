Posibilidades de comandos para crear commit y hacer push 
Desde la terminal con Git (la forma más común)

Pasos típicos:
git add .
git commit -m "mensaje del commit"
git push

git add → prepara los archivos

git commit → guarda el cambio en el historial

git push → envía los cambios a GitHub

También se puede agregar un archivo específico:

git add archivo.txt
git commit -m "cambio en archivo"
git push
---
# Comandos del profe

`gh auth login` : se loguea a github de forma semi automatica

`gh repo clone` : <repositorio>: Clona el repositorio indicado
`git add .` : Agrega archivos a la etapa de staging
