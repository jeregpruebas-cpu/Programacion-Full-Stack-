<html> </html> -> Es el contenedor principal, el cuál guarda todas las etiquetas correspondientes de la pagina

<a></a> -> Permite añadir vinculos o redirecciones a distintas etiquetas del html hacia vinculos web o archivos locales

<h1></h1> -> <h6></h6> -> Permiten añadir titulos de distinto tamaño

<p></p> -> Permite añadir parrafos de texto

<img/> -> Permite añadir imagenes a la pagina

<br> -> Salto de linea

<header></header> -> Es la cabecera de la pagina

<footer></footer> -> Es el pie de la pagina

<table></table> -> Permite crear tablas
    <tr></tr> -> Permite añadir una fila a la tabla correspondiente
    <th></th> -> Permite añadir filas que actuan como cabecera
 

    <details>
        <summary>
        "Texto que se despliega"
        </summary>
    </details>
